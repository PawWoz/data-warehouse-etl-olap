# Finance Data Warehouse Project

This project involves designing and implementing a **Data Warehouse** for financial and time tracking data. It includes the full data flow — from a Microsoft Access source database to an OLAP cube — using tools like SQL Server, SSIS, and SSAS.

## 📌 Project Scope

1. Extend a source database (`Finanse i Czas.accdb`) to track time in addition to expenses and income.
2. Populate the source database with sample data for several users.
3. Design a conceptual model of the data warehouse.
4. Create logical models in both **star schema** and **snowflake schema**.
5. Implement the data warehouse in **SQL Server**.
6. Design and implement an **ETL process** using **SSIS**.
7. Populate the data warehouse from the source database.
8. Publish **OLAP cubes** using **SSAS**.
9. Perform analytical queries on the OLAP cubes (Excel or other BI tools).

## 🔧 Tools Used

- Microsoft Access
- Microsoft SQL Server
- SQL Server Integration Services (SSIS)
- SQL Server Analysis Services (SSAS)
- Microsoft Excel

## 📈 Sample Analyses

Analyses performed on OLAP cubes include:
- Total and average expenses by category and user
- Time spent per activity and day
- Income vs expenses per user over time
- Aggregated monthly reports
